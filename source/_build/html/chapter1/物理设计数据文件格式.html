设计交换格式 def 文件¶

设计交换格式 def 文件：

设计交换格式（ def ， design exchange format）文件是展平式的设计网表文件，是不同的 EDA工具交互过程中的主流数据交换格式

设计交换格式 def 文件是一种带有信息的网表，除了包括 Verilog 所包含的连接信息外，还有各个单元所存在的物理信息 —— 定义了芯片的面积（布图规划后的记录）、布图规划区域定义、电压域定义（电源规划后的记录）、标注单元行、标准单元的位置属性、端口位置信息（布局后的记录）、布线的内容、网线 Net 的连接以及一些连接网线的连接（布线后的记录）

应用示例：

  • 在布局阶段根据不同的需求导出相应的布图规划、布局信息，从而实现数据交互

  • 在电源规划时，有多个电压域，分别将各个电压域的电源线导出为分立的 def ，再读入设计
