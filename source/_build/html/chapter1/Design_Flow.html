Implementation¶

Synthesis -> Floorplanning -> Placement -> CTS -> Routing

从布图规划开始，到电源规划到完成布局，每一步规划不仅仅要考虑布线的实现，还要在布局完成之后时提供好的数据环境，供时钟树综合使用

时序¶
