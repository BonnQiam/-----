设计约束文件格式sdc.rst¶

设计约束文件格式 sdc （Synopsys Design Constraints） 文件基于 TCL （tool command language ）格式，由 Synopsys 开发定义，起初用于电路的逻辑综合。1998年由 Cadence 公司移植用于时序控制的布局布线设计

警告

sdc 文件针对电路的时序、功耗、面积等方面进行约束，从而使得芯片满足设计要求的规范。
